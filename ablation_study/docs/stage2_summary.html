Stage 2: 반도체 공급 신호 → 주가 수익률 예측 검증

대상 종목SK하이닉스(메모리), ASML(장비) 분석 기간2004-12 ~ 2023-07 (월별, 224개월) 검증 방식Ablation Study (Model A/B/C) + Walk-forward Cross-Validation

1. Overview

Stage 1에서 WSTS 글로벌 반도체 매출 YoY%의 t+6 예측값(wsts_pred_t6)을 생성했다. Stage 2는 이 공급 사이클 예측값이 실제 반도체 기업 주가 수익률의 유의미한 선행 신호인지를 검증한다.

주가 6개월 forward return을 타겟으로 선택한 이유는 두 가지다. 첫째, 분기 매출이나 EPS 같은 실적 데이터는 발표 주기가 길고 revision이 잦아 월별 시계열 모델에 연결하기 어렵다. 반면 주가는 시장 참여자들의 미래 기대를 실시간으로 반영하는 고주파 신호다. 둘째, h=6(6개월)은 Stage 1 예측 horizon과 정합한다. Stage 1이 "지금부터 6개월 후 업황"을 예측하므로, Stage 2에서도 6개월 후 주가 수익률로 공급 신호의 실효성을 검증하는 것이 논리적으로 일관된다.

검증 방식은 Ablation Study다. 공급 신호(wsts_pred_t6)를 포함한 풀 모델(A)과 제외한 모델(B), 공급 신호만 사용한 모델(C)을 비교해 공급 신호의 incremental 기여도와 독립적 설명력을 정량화한다.

2-2. OOS Prediction — Leakage 방지 설계

wsts_pred_t6를 단순히 Stage 1 모델의 전체 데이터로 학습한 예측값으로 사용하면 데이터 누출(leakage)이 발생한다. 미래 정보가 학습에 스며들어 검증 성능이 과대평가되기 때문이다.

이를 막기 위해 Expanding-window Walk-forward로 생성한 Out-of-Sample(OOS) 예측값만 사용한다. 각 시점 t에서 t 이전의 데이터만으로 학습한 모델의 예측값으로 wsts_pred_t6를 구성함으로써, 모든 시점의 예측값이 그 시점에서 실제로 관측 가능한 정보에만 의존한다.

t=60: train[0:60]  → predict t=60
t=61: train[0:61]  → predict t=61
t=62: train[0:62]  → predict t=62
...
t=n:  train[0:n]   → predict t=n

이 OOS 예측 시퀀스를 Stage 2의 wsts_pred_t6 피처로 사용하기 때문에, Stage 2의 walk-forward CV 역시 같은 원칙 하에 미래 정보 없이 검증이 이루어진다.

  • 공급 신호의 독립적 기여 확인: 환율(USD/KRW)을 통제한 뒤에도 wsts_pred_t6의 incremental 기여는 RMSE −1.01, DirAcc +4.3%p로 명확하게 존재한다. 공급 신호가 환율 효과를 반영하는 것이 아닌, 별도 정보를 제공하고 있음을 시사한다.

4. ASML 분석

4-1. 왜 ASML인가?

SK하이닉스(메모리 반도체 제조사)에서의 검증만으로는 "공급 사이클 신호가 메모리 업종에만 유효한가"라는 질문에 답할 수 없다. ASML은 반도체 노광 장비(EUV·DUV)의 독점적 공급자로, 반도체 제조사의 생산 능력 투자 결정에 직접 연결된다. 메커니즘 가설은 단순하다: "공급 확장이 예측되면 → 반도체 제조사가 설비 투자를 늘리고 → ASML에 장비 발주가 증가한다." 이 경로가 유효하다면, Stage 1의 공급 예측값이 메모리 제조사뿐 아니라 장비 섹터 주가에도 선행 신호로 작동해야 한다. 서로 다른 섹터 포지션에서 동일한 공급 신호의 유효성을 검증함으로써 분석의 일반성을 높인다.

4-4. Model C 해석 — 공급 신호의 독립 설명력

Model C(공급 신호 단독)의 DirAcc 67.1%는 다음과 같이 해석한다.

  • 랜덤 기준선(50%) 대비 +17%p: 공급 신호 하나만으로 ASML 주가의 상승/하락 방향을 랜덤보다 17%p 더 정확히 예측한다. 단일 피처 기준으로 유의미한 설명력이다.
  • Model B(78.0%) 대비 −11%p: 거시경제·FX 등 다른 피처들이 더 강한 설명력을 보유하고 있으므로, 공급 신호가 "지배적 신호"라고는 할 수 없다. 하지만 "독립적으로 유효한 신호"임은 분명하다.
  • C → A로 갈 때 +12%p 개선: 공급 신호를 다른 거시 피처와 결합할 때 시너지가 발생한다. 이는 공급 신호가 거시 환경 정보를 일부 내재하고 있어, 공급·수요 양측 정보를 동시에 활용할 때 예측 정확도가 높아짐을 시사한다.

4-6. ASML 핵심 발견

  • RMSE+DirAcc 동시 양의 기여 확인: 종목 전용 EUR 환율 피처 추가 후 ΔRMSE −0.32, ΔDirAcc +1.2%p로 두 지표 모두 양의 기여를 보인다. SK하이닉스 대비 절대 기여폭은 작지만, 방향성은 일관된다.
  • 장비 섹터에서도 공급 사이클 선행 신호 유효: ASML은 메모리 제조사가 아닌 장비 공급자로, 공급 신호가 단순히 "메모리 업황 → 하이닉스 매출"의 직접 연결이 아닌, 산업 사이클 전반에 걸친 선행 정보를 포함함을 시사한다.
  • Model C 단독 설명력 랜덤 대비 +17%p: 공급 신호 하나만으로 67.1%의 방향 정확도를 달성한다. 이는 wsts_pred_t6가 포트폴리오 관점에서 독립적으로 활용할 수 있는 신호 품질을 보유하고 있음을 보여준다.

  • 포지션이 달라도 일관된 방향: SK하이닉스에서는 공급 신호가 SHAP 2위의 주요 드라이버로 작동하고, ASML에서는 보완적 정보원으로 기여한다. 기여 강도의 차이는 있지만 방향은 동일하며, 이는 Stage 1 공급 예측 모델의 산업 전반에 걸친 선행 지표로서의 가능성을 보여준다.
